# voitta-misc

Catch-all plugin for Voitta-curated Claude Code content.

As shared content grows, themed plugins (`voitta-aws`, `voitta-git`,
`voitta-debug`, `voitta-infra`) will split out of here. See
`../../docs/architecture.md`.

## Install

```
/plugin marketplace add voitta-ai/omemepo
/plugin install voitta-misc
```
